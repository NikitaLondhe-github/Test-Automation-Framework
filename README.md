# Test-Automation-Framework
# Selenium Test Automation Framework

## Tech Stack

- Java
- Selenium WebDriver
- TestNG
- Maven
- Page Object Model
- Log4j2
- Extent Reports

## Features

- Cross-browser support
- Screenshot on failure
- Logging
- Custom Exceptions
- TestNG Listener
- Configurable through application.properties

## Run the Tests

mvn clean test
